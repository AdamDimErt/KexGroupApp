    **2. Add 4 report methods to dashboard.service.ts** (add at end of class, before closing brace):

    **getReportDds(tenantId: string, dateFrom: string, dateTo: string):**
    - Get restaurants for tenant: `this.prisma.restaurant.findMany({ where: { brand: { company: { tenantId } }, isActive: true }, select: { id: true, name: true } })`
    - Get expenses grouped by restaurantId + articleId: `this.prisma.expense.groupBy({ by: ['restaurantId', 'articleId'], where: { restaurantId: { in: restaurantIds }, date: { gte: startDate, lte: endDate } }, _sum: { amount: true } })`
    - Fetch articles with groups: `this.prisma.ddsArticle.findMany({ where: { id: { in: articleIds } }, include: { group: { select: { name: true } } } })`
    - Build Map<restaurantId, Map<groupName, amount>> from expense rows
    - For each restaurant: compute totalExpense = sum of all groups, compute share = group amount / totalExpense * 100
    - Compute totals.totalExpense = sum of all restaurant totals
    - Return DdsReportDto shape

    **getReportCompanyExpenses(tenantId: string, dateFrom: string, dateTo: string):**
    - Query: `this.prisma.expense.groupBy({ by: ['articleId', 'source'], where: { restaurantId: null, date: { gte: startDate, lte: endDate } }, _sum: { amount: true } })`
    - IMPORTANT: use `restaurantId: null` (not undefined) to find unallocated HQ expenses
    - Fetch article names: `this.prisma.ddsArticle.findMany({ where: { id: { in: articleIds } }, select: { id: true, name: true } })`
    - Compute totalAmount = sum of all category amounts
    - For each category: share = (amount / totalAmount) * 100, round to 2 decimal places
    - Return CompanyExpensesReportDto shape

    **getReportKitchen(tenantId: string, dateFrom: string, dateTo: string):**
    - Purchases: `this.prisma.kitchenPurchase.findMany({ where: { tenantId, date: { gte: startDate, lte: endDate } }, orderBy: { date: 'asc' } })`
    - Map to: `{ date: p.date.toISOString(), description: p.productName, amount: this.toNumber(p.amount) }`
    - Get restaurant IDs for tenant (same pattern as DDS)
    - Shipments grouped: `this.prisma.kitchenShipment.groupBy({ by: ['restaurantId'], where: { restaurantId: { in: restaurantIds }, date: { gte: startDate, lte: endDate } }, _sum: { amount: true }, _count: { id: true } })`
    - Map shipments to rows with restaurant name lookup
    - Compute totals: totalPurchases = sum of purchase amounts, totalShipments = sum of shipment amounts
    - Return KitchenReportDto shape

    **getReportTrends(tenantId: string, dateFrom: string, dateTo: string):**
    - Get restaurant IDs for tenant
    - Daily revenue: `this.prisma.financialSnapshot.groupBy({ by: ['date'], where: { restaurantId: { in: restaurantIds }, date: { gte: startDate, lte: endDate } }, _sum: { revenue: true }, orderBy: { date: 'asc' } })`
    - Daily expenses: `this.prisma.expense.groupBy({ by: ['date'], where: { restaurantId: { in: restaurantIds }, date: { gte: startDate, lte: endDate } }, _sum: { amount: true }, orderBy: { date: 'asc' } })`
    - Merge by date key: create Map<dateString, {revenue, expenses}>, iterate both arrays
    - For date formatting: use `.toISOString().slice(0, 10)` to get YYYY-MM-DD from Date object
    - Compute netProfit = revenue - expenses per day
    - Compute summary: avgDailyRevenue = totalRevenue / points.length (or 0 if empty), avgDailyExpenses = totalExpenses / points.length, totalNetProfit = totalRevenue - totalExpenses
    - Return TrendsReportDto shape

    All 4 methods: use `this.parseStartDate(dateFrom)` and `this.parseEndDate(dateTo)` for timezone. Use `this.toNumber()` for all Decimal values.
