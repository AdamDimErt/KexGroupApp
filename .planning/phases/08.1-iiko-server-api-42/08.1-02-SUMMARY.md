## Decisions Made
- GROUP_COLORS inlined in each screen (not extracted to shared constants file) — mobile scope is self-contained, avoids premature abstraction
- Emoji icons used per explicit user request ("красиво") — exception to no-emoji rule documented in code comments
- Simple show/hide accordion (no animation) — better performance on low-end Android devices with many restaurant sub-rows
- expandedGroup stores groupId string for natural single-accordion pattern; null = all collapsed
